# Watermelon
![image](https://user-images.githubusercontent.com/23046289/230736220-e9045166-7596-4067-be61-716b666ef90d.png)


## 模板
2D

## 内容
- 背景
- 墙壁
- 水果
- 死亡线
- 分数
- 音乐

## 逻辑
- 待命水果，鼠标按下时在 X 轴跟随鼠标，鼠标谈起时下落
- 水果落地、水果相互间有碰撞（带弹性），相同水果碰撞到一起会合成更大的水果
- 水果抵达死亡线时，游戏失败
- 显示当前分数

## 主要技术
- 碰撞器
- 刚体（重力）
- 鼠标事件
- 音乐播放
- 文字显示
